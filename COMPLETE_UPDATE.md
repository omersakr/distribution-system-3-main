## 🚀 كيفية الاستخدام

### للودرز التلقائية:
```javascript

async function loadData() {
    showInlineLoader('containerId', 'جاري التحميل...');
    try {
        const data = await apiGet('/endpoint');
        renderData(data);
    } catch (error) {
        showError(error);
    }
}
```
